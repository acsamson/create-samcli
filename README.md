# Intro

yarn create-samcli 包名

samcli 脚手架.

如果想要修改本 package 需要进行 `yarn link`, 使用 `create-samcli 你的包名称` 即可生成你要的包

# Use

```
yarn create-samcli 包名

? 请选择一个模板类型 (Use arrow keys)
❯ samcli-pro - 🎃 选择一个中台模板

? 🍧 请选择要使用的语言?（推荐使用TS (Use arrow keys)
❯ TypeScript
  JavaScript

? 💁‍♂️ 想使用哪种脚手架? (Use arrow keys)
❯ 魔改版
```
